## 库操作

| 语法                           | 作用                   |
| ------------------------------ | ---------------------- |
| net start mysql                | 启动 mysql 服务        |
| net stop mysql                 | 停止 mysql 服务        |
| mysql -u root -p               | 连接(登录) mysql       |
| exit                           | 退出连接               |
| show databases;                | 查看所有数据库         |
| create database 数据库名;      | 创建数据库             |
| show create database 数据库名; | 查看创建数据库时的语句 |
| drop database 数据库名;        | 删除数据库             |
| use 数据库名;                  | 选择数据库             |
